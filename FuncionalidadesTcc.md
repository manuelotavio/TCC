-----------------------------------------------

<center>Tema: Sistema de abertura de chamados.</center>
--------------------------------------

###<u>Funcionalidades:

* Senha para acessar o sistema.
* Criação de usuários.
* Abertura de chamados.
* Gerar numero de protocolo ou do chamado.
* Salvar todos os chamados.
* Possibilidade de imprimir o chamado.
* Listar todos os chamados pendentes ou que precisam de retorno.
* Gerar graficos dos chamados.


###<u>Tecnologias:

* Será um sistema WEB.
* banco de dados MySql.
* java para fazer as funcionalidades do site.
