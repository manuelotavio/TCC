Trabalho de conclusão de curso.

Tema: Sistema de abertura de chamados.

Funcionalidades:
1) Senha para acessar o sistema.
2) Criação de usuarios.
3) Abertura de chamados.
4) Gerar numero de protocolo ou do chamado.
5) Salvar todos os chamados.
6) Possibilidade de imprimir o chamado.
7) Listar todos os chamados pendentes ou que precisam de retorno.
8) Gerar graficos dos chamados.


Tecnologias:
1) Será um sistema WEB.
2) Vou usar banco de dados MySql.
3) Vou usar java para fazer as funcionalidades do site.
